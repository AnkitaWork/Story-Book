# Welcome to [**Social Authentication**](https://www.npmjs.com/package/angularx-social-login)

 Login with Facebook and Google makes it safe and easy for users to use applications. When a user clicks on the Login button with Facebook or Gmail, the user is navigated to Facebook or Google to give the app permission. In response, the user receives a Token key and other personal details.

[**Social login and authentication module**](https://www.npmjs.com/package/angularx-social-login) for Angular 9 / 10. Supports authentication with [**Google**](https://developers.google.com/api-client-library/javascript/reference/referencedocs#gapiauth2clientconfig), [**Facebook**](https://developers.facebook.com/docs/reference/javascript/FB.login/v8.0), and [**Amazon**](https://developer.amazon.com/).

Installing via npm
`npm i angularx-social-login`
