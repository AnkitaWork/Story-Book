# Welcome to [**Toast**](https://www.npmjs.com/package/ngx-toastr)

Install `npm install ngx-toastr --save`

`@angular/animations` package is a required dependency for the default [**toast**](https://www.npmjs.com/package/ngx-toastr)

Toast Component Injection without being passed ViewContainerRef

No use of `*ngFor.` Fewer dirty checks and higher performance

AoT compilation and lazy loading compatible

Component inheritance for custom toasts

SystemJS/UMD rollup bundle

Animations using Angular's [**Web Animations API**](https://angular.io/guide/animations)

Output toasts to an optional target directive

[**DEMO**](https://ngx-toastr.vercel.app/)
